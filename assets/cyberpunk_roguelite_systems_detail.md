## 第二部分：技能系统（Skills）

### 2.1 技能分类
- **主动技能**：玩家主动释放的技能，消耗法力值
- **被动技能**：自动生效的属性加成或机制增强
- **终极技能**：充能完成后释放的强力技能，每场战斗限制使用次数

### 2.2 街头武士技能树

#### 主动技能（分3条分支）

**分支1：进攻型（伤害强化）**
- 技能1：**横扫斩**（1级解锁）
  - 消耗：20法力
  - 效果：前方扇形范围斩击，造成150%武器伤害
  - 升级效果：
    - +1级：范围扩大20%
    - +2级：伤害提升至180%武器伤害
    - +3级：击退效果
    - +4级：击中敌人后，下一次横扫斩必定暴击

- 技能2：**旋风斩**（分支前置技能：横扫斩+2）
  - 消耗：40法力
  - 效果：原地旋转攻击，持续2秒，对周围敌人造成200%武器伤害/秒
  - 升级效果：
    - +1级：持续时间延长至2.5秒

    - +3级：每次旋转积累振动槽，满槽后释放冲击波

- 终极技能：**破碎虚空**（分支前置技能：旋风斩+3）

## 第三部分：合成系统（Crafting）

### 3.1 基础合成规则

#### 同品级升级
- 2个普通物品 → 1个稀有物品
- 2个稀有物品 + 5齿轮 → 1个史诗物品
- 2个史诗物品 + 3芯片 → 1个传说物品
- 2个传说物品 + 1核心晶体 → 1个终极物品

#### 异类合成（创造新物品）
- 武器 + 武器 = 双属性融合武器
- 防具 + 防具 = 复合防具
- 武器 + 材料 = 强化版武器
- 防具 + 材料 = 强化版防具

- **技能预览**：鼠标悬停技能节点时，显示技能详细信息（名称、效果、升级效果）

#### 技能树交互
- **点击解锁**：点击技能节点进行解锁，消耗对应技能点
- **重置技能点**：提供"重置技能点"功能，重置后返还所有已消耗技能点（仅在安全区域可用）
- **技能预设**：支持保存多个技能预设，方便快速切换不同技能搭配

## 第五部分：数值平衡参考

### 5.1 伤害计算公式

**基础伤害** = 武器攻击力 × 技能倍率 × 暴击倍率（1.5x）× 元素克制倍率

**元素克制倍率**：
- 物理 → 生物：1.2x
- 物理 → 机械：0.8x
- 能量 → 生物：1.0x
- 能量 → 机械：1.5x
- 生化 → 生物：0.5x
- 生化 → 机械：1.2x

### 5.2 护甲减伤公式

**实际伤害** = 原始伤害 × (100 / (100 + 护甲值))

示例：
- 护甲值为50时，减伤约33%
- 护甲值为100时，减伤约50%
- 护甲值为200时，减伤约67%

### 5.3 暴击率计算

**基础暴击率** = 5% + 职业基础暴击率 + 装备加成 + 技能加成

**暴击伤害** = 150% + 装备加成 + 技能加成

### 5.4 经验值与等级系统
