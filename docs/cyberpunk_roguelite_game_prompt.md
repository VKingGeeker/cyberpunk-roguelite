# 赛博朋克实时动作肉鸽游戏开发提示词

## 项目核心定位

**类型**：实时动作肉鸽（Real-time Action Roguelite）
**视觉风格**：手绘赛博朋克（Hand-drawn Cyberpunk）
**技术栈**：Phaser 3 + TypeScript + Vite
**游玩节奏**：割草式持续战斗，升级获得新技能
**核心钩子**：被动技能自动释放 + 武器合成 + 大世界探索

---

## 一、核心玩法循环

### 1.1 世界结构（MVP已实现）
```
大世界地图（3200x2400）= 持续战斗区域
玩家在开放世界中与不断生成的敌人战斗
敌人在玩家周围环形区域生成，持续追踪玩家
相机跟随玩家移动，无固定关卡限制
```

### 1.2 战斗系统

#### MVP已实现设计
- **被动技能系统**：
  - 技能自动释放，无需手动操作和法力消耗
  - 每个技能有独立冷却时间，冷却结束后自动触发
  - 升级时随机选择3个技能（新技能/已有技能升级）
  - 降低操作门槛，强调策略选择和爽快感

- **武器系统**：
  - 通用武器系统，不按职业分类
  - 3个武器槽位，按键快速切换
  - 武器属性直接影响攻击力、攻速、暴击等
  - 12种武器（普通3/稀有3/史诗3/传说3）

- **伤害计算**：
  - 攻击力 = 玩家基础攻击 + 武器攻击加成
  - 暴击率 = 基础暴击 + 武器暴击加成
  - 伤害浮动范围：±10%
  - 暴击伤害倍率：默认150%，可被武器加成

- **战斗节奏**：
  - 大量敌人持续生成，割草式体验
  - 普通怪：快速清理，积攒连击数
  - 精英怪：更高血量，掉落更好奖励
  - BOSS：阶段式攻击模式，弱点暴露窗口期

#### 正式版设计目标
- **基础机制**：
  - 近战武器 + 远程技能双切，流畅连招系统
  - 受击判定区精确定位，强调闪避和格挡时机
  - 伤害计算考虑护甲穿透、元素克制（物理/能量/生化）

### 1.3 时间回溯机制（正式版）
- **触发条件**：
  - 生命值归零时，消耗"时空碎片"触发
  - 或在战斗中主动激活（冷却时间：3分钟）

- **回溯效果**：
  - 单人模式：倒流至最近存档点（保留物品和经验）
  - 联机模式：队友可投票复活阵亡玩家，消耗全队时空碎片

- **限制**：
  - 每次回溯扣除20%当前收益（经验/金钱/材料）
  - 每关最多回溯2次，超过直接失败

## 二、物品合成系统

### 2.1 MVP已实现设计
- **基础合成公式**：
  - 2件同品质武器 → 1件更高品质武器
  - 合成结果随机从对应品质池中选取
  
- **合成界面**：
  - 显示可用配方列表
  - 材料需求检查
  - 合成结果预览

### 2.2 正式版设计目标
- **特殊公式**：
  - 武器 + 武器 = 双属性融合武器（如：热能武士刀 + 电磁匕首 = 热能电磁双刃）
  - 装备 + 材料 = 属性强化版（如：赛博臂甲 + 纳米纤维 = 轻量化赛博臂甲）
  - 特定组合 = 隐藏技能解锁（详见2.3）

### 2.3 材料体系
| 材料类型 | 来源 | 用途 |
|---------|------|------|
| 齿轮 | 普通怪掉落 | 基础合成 |
| 芯片 | 精英怪掉落 | 中级合成 |
| 核心晶体 | BOSS掉落 | 高级合成 |
| 黑暗物质 | 随机事件、隐藏区域 | 传说级合成 |

### 2.4 隐藏技能组合示例（正式版）
- **组合1**：火焰喷射器 + 冰霜护手 = **蒸汽爆破**（范围AOE + 眩晕）
- **组合2**：时间减速装置 + 速度靴 = **时空闪现**（短距离瞬移 + 时间延迟伤害）
- **组合3**：医疗纳米机器人 + 毒气手雷 = **生化炼金**（对敌人释放毒素，对自己释放治疗）

## 三、多职业与技能树

### 3.1 MVP已实现设计

#### 被动技能系统
- **技能分类**：攻击型 / 防御型 / 辅助型
- **技能总数**：17种
- **触发机制**：自动触发，无手动操作
- **冷却机制**：每个技能独立冷却

#### 已实现技能列表
**攻击型（10种）**：
| 技能名 | 效果 | 冷却 |
|--------|------|------|
| 霓虹斩击 | 前方扇形斩击，150%伤害 | 2秒 |
| 等离子漩涡 | 周围AOE，200%伤害 | 5秒 |
| 连锁闪电 | 链式攻击，最多3目标，120%伤害 | 3秒 |
| 激光射线 | 直线贯穿，180%伤害 | 4秒 |
| 等离子球 | 追踪弹，160%伤害 | 3秒 |
| 能量新星 | 大范围AOE，250%伤害 | 8秒 |
| 音爆冲击 | 击退敌人，140%伤害 | 4秒 |
| 烈焰波 | 火焰波，130%伤害+燃烧 | 3秒 |
| 虚空裂缝 | 持续伤害区域，80%/秒 | 6秒 |
| 冰霜碎片 | 冰锥攻击，冻结效果 | 4秒 |

**防御型（2种）**：
| 技能名 | 效果 | 冷却 |
|--------|------|------|
| 纳米护盾 | 挡下一次攻击，回复10%生命 | 12秒 |
| EMP冲击 | 眩晕周围敌人2秒 | 8秒 |

**辅助型（5种）**：
| 技能名 | 效果 | 冷却 |
|--------|------|------|
| 超频驱动 | 攻速和移速提升30%，持续5秒 | 10秒 |
| 全息幻影 | 创建诱饵吸引敌人 | 8秒 |
| 时间扭曲 | 减速周围敌人50%，持续3秒 | 12秒 |
| 纳米虫群 | 持续治疗，每秒5%生命，持续5秒 | 15秒 |
| 能量汲取 | 攻击时吸取敌人生命 | 被动 |

#### 升级机制
- 击杀敌人获得经验值
- 升级时弹出选择界面，随机3个技能
- 新技能：直接学习
- 已有技能：升级（最高5级）
- 升级效果：伤害提升10%、范围扩大、冷却减少10%

### 3.2 正式版职业体系
| 职业 | 核心定位 | 初始武器 | 被动技能 |
|------|---------|---------|---------|
| 街头武士 | 近战爆发 | 高频振动刀 | 每次暴击回复生命 |
| 数据黑客 | 远程控制 | 神经干扰枪 | 技能暴击造成额外伤害 |
| 生化改造者 | 坦克/辅助 | 生物注射枪 | 受击时获得临时护盾 |
| 暗影刺客 | 潜行/爆发 | 隐形匕首 | 背刺必定暴击 |

### 3.3 正式版技能树结构
- **通用技能树**（所有职业共享）：
  - 移动速度提升、闪避冷却缩减、拾取范围扩大
  - 时间回溯效果增强（减少收益扣除）

- **职业专属技能树**：
  - 分为3条分支：进攻型 / 防御型 / 功能型
  - 每条分支5个节点，最终技能为该分支终极能力

- **技能升级**：
  - 使用技能经验点（从关卡完成奖励获得）
  - 每次升级可在3个随机选项中选择（类似《哈迪斯》）

## 四、随机事件系统（正式版）

### 4.1 地图事件类型
| 事件类型 | 描述 | 奖励 | 风险 |
|---------|------|------|------|
| 黑市商人 | 出售稀有物品和配方 | 购买机会 | 价格高昂，可能买到假货 |
| 陷阱房间 | 满满的机关和敌人 | 大量材料和隐藏宝箱 | 高死亡率 |
| 神龛祭坛 | 献祭当前装备换取祝福 | 稀有属性加成 | 永久失去献祭物品 |
| 幸运转盘 | 投掷骰子决定结果 | 双倍收益或一无所有 | 纯随机 |
| 幸存者营地 | NPC互动任务 | 道具/信息/技能点 | 可能遭遇敌对势力突袭 |

### 4.2 事件触发算法
- 每关地图随机生成3-5个事件点
- 事件类型权重：
  - 商人：30%
  - 陷阱：25%
  - 神龛：20%
  - 转盘：15%
  - 营地：10%

## 五、联机与本地模式（正式版）

### 5.1 联机模式
- **人数**：2-4人合作
- **难度调整**：玩家数量越多，敌人血量和攻击力提升（非线性，避免过难）
- **共享机制**：
  - 掉落物品按职业需求分配
  - 经验值共享，等级同步
  - 时间回溯投票制

### 5.2 本地模式
- **单人体验**：完整游戏内容
- **存档系统**：
  - 自动存档（每个事件点后）
  - 手动存档（仅限安全区域）
  - 死亡惩罚（按难度设置）

### 5.3 数据同步
- 联机模式下的进度可同步至本地模式
- 独占奖励（联机专属装备/皮肤）需在线获取

## 六、视觉与音效设计

### 6.1 视觉风格
- **手绘赛博朋克**：
  - 霓虹配色：青色、品红、紫色为主，高对比度
  - 机械与肉体融合的设计语言
  - 背景细节：全息广告牌、飞行载具、贫民窟摩天楼

- **角色设计**：
  - 每个职业有鲜明的赛博改造特征
  - 装备外观实时变化（不同材质和光效）

- **MVP实现**：
  - 程序生成角色纹理（Idle/Run/Attack动画帧）
  - 霓虹发光效果
  - 伤害数字显示（暴击高亮）

### 6.2 UI/UX
- **HUD设计**：极简主义，关键信息突出显示
  - 生命条（赛博朋克风格）
  - 武器栏（3槽位显示）
  - 技能栏（已学技能图标+冷却指示）
  - 经验条和等级显示
  - 击杀计数器

- **合成界面**：拖拽式操作，实时预览合成结果
- **技能选择界面**：升级时3选1，显示技能详情

### 6.3 音效设计
- **战斗音效**：金属碰撞、能量爆发、电子故障声
- **环境音效**：霓虹灯电流声、远处的爆炸、警笛
- **音乐**：赛博朋克电子乐，战斗时节奏加快

## 七、技术实现要点

### 7.1 游戏引擎选择
- **当前实现**：Phaser 3.90.0 + Vite 5.4 + TypeScript 5.9
- **联机方案**（正式版）：Socket.io + Node.js后端
- **状态同步**：客户端预测 + 服务器权威校验

### 7.2 程序化生成
- **地图生成**：
  - MVP：固定大世界地图（3200x2400）
  - 正式版：基于瓦片（Tile）的随机迷宫算法
  - 事件点和宝箱位置随机分布
  - 主题变化（每关不同视觉主题：贫民窟→工业区→企业总部）

- **敌人AI**：
  - MVP：简单追踪AI，纯近战攻击
  - 正式版：行为树（Behavior Tree）设计
  - 根据玩家数量和难度动态调整AI参数

### 7.3 性能优化
- 对象池管理敌人和特效
- LOD（Level of Detail）系统
- WebGL加速渲染
