## 第三部分：技能系统（MVP已实现）

### 3.1 技能设计理念
MVP版本采用**被动技能自动触发**机制，降低操作门槛，提升爽快感。

### 3.2 技能分类
- **攻击型技能**：造成伤害，清怪主力
- **防御型技能**：保护玩家，控制敌人
- **辅助型技能**：增益效果，提升战斗力

### 3.3 技能触发机制
```typescript
// 技能自动触发逻辑
interface SkillData {
    skill: Skill;
    level: number;
    cooldownEndTime: number;
}

// 每帧检查
this.ownedSkills.forEach((data, skillId) => {
    if (time < data.cooldownEndTime) return;  // 冷却中
    if (enemies.length === 0) return;         // 无敌人
    
    this.triggerSkill(skillId, skill, enemies, time);
    data.cooldownEndTime = time + skill.cooldown * 1000;
});
```

## 第四部分：合成系统

### 4.1 MVP合成系统（已实现）

#### 基础合成规则
| 合成公式 | 结果 |
|---------|------|
| 普通武器 × 2 | 稀有武器（随机） |
| 稀有武器 × 2 | 史诗武器（随机） |
| 史诗武器 × 2 | 传说武器（随机） |

#### 合成界面功能
- 配方列表显示
- 材料需求检查
- 合成结果预览
- 一键合成按钮

### 4.2 正式版合成系统（规划中）

#### 同品级升级
- 2个普通物品 → 1个稀有物品
- 2个稀有物品 + 5齿轮 → 1个史诗物品
- 2个史诗物品 + 3芯片 → 1个传说物品
- 2个传说物品 + 1核心晶体 → 1个终极物品

#### 异类合成（创造新物品）
- 武器 + 武器 = 双属性融合武器
- 防具 + 防具 = 复合防具
- 武器 + 材料 = 强化版武器
- 防具 + 材料 = 强化版防具

### 4.3 特殊合成配方（正式版隐藏技能解锁）

#### 武器融合配方

**融合公式3：医疗纳米机器人 + 毒气手雷 = 生化炼金**
- 属性：治疗 + 毒素
- 隐藏技能：**生化炼金**（对敌人释放毒素，对自己释放治疗，持续10秒）

**融合公式4：激光枪 + 电磁炮 = 粒子光束**

**融合公式5：振动短刀 + 隐形匕首 = 幽灵刃**

**融合公式6：数据切割刀 + 生化利爪 = 赛博生化刃**
